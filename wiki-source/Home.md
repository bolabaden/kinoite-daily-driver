# Kinoite daily driver wiki

Documentation for **[bolabaden/kinoite-daily-driver](https://github.com/bolabaden/kinoite-daily-driver)** (Fedora Kinoite / WSL2 workspace).

- **Repository** — [github.com/bolabaden/kinoite-daily-driver](https://github.com/bolabaden/kinoite-daily-driver)
- **Jekyll sources** — live in this wiki clone (`index.md`, `_docs/`, `_config.yml`). Run `bundle install` then `bundle exec jekyll serve` locally.
- **Sync from main repo** — in the main checkout run `.\scripts\Kinoite-Wiki.ps1 -Action Sync` to refresh mirrored pages and push.

After enabling **GitHub Pages** on this wiki repository, the built site URL will appear under repository **Settings → Pages**.
