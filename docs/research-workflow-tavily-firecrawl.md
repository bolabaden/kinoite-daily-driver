# Research workflow (Tavily / Firecrawl / web)

- Prefer **primary sources** (Fedora Magazine, docs.fedoraproject.org, Flathub app pages) over random blogs for **install** commands.
- When using automated web tools, store **digests** under `.firecrawl/` with **date-stamped** filenames; **sanitize** before sharing.
- If **`tvly` / Tavily CLI** is not installed on the Windows host, add digests under **`../research/`** (committed) and cite URLs — see e.g. `../research/kinoite-wsl-systemd-sources-2026-04-25.md`. Install when available: [Tavily CLI](https://github.com/tavily-ai/tavily) / `tvly` per project skill.
